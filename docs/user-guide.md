# User Guide

## Registration

1. Open the application.
2. Navigate to Register.
3. Create an account.

## Login

1. Enter email and password.
2. Press Login.

## Add Clothing Item

1. Select Add Item.
2. Upload an image.
3. Remove the background if desired.
4. Select:
   - Type
   - Color
   - Style
5. Save the item.

## Modify Clothing Item

1. Open the wardrobe.
2. Select a garment.
3. Edit its information.
4. Save the changes.

## Delete Clothing Item

1. Open the garment details.
2. Press Delete.
3. Confirm deletion.

## Generate Outfit

### Today's Outfit

1. Open Create Outfit.
2. Select Today.
3. Enter a location.
4. Generate the recommendation.

### Tomorrow's Outfit

1. Select Tomorrow.
2. Enter a location.
3. Generate the recommendation.

### Weekly Outfit

1. Select Week.
2. Enter a location.
3. Review the generated outfits.

## Logout

1. Open the user menu.
2. Select Logout.

## Troubleshooting

### Image upload fails

Verify that the selected file is a valid image format.

### Background removal fails

Verify that the uploaded image is not corrupted.

### Outfit generation fails

Verify that:

- The wardrobe contains clothing items.
- The selected location is valid.
- The weather service is available.

### Authentication problems

Verify that:

- The user credentials are correct.
- The account exists.
- The authentication service is available.
